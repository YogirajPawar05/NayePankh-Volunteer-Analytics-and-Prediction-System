# NayePankh Volunteer Analytics & Prediction System

## Project Overview

The NayePankh Volunteer Analytics & Prediction System is a Data Analytics, Machine Learning, and Artificial Intelligence project developed to help NayePankh Foundation improve volunteer management and decision-making.

The project analyzes volunteer participation patterns, identifies engagement trends, predicts future volunteer participation, and provides AI-based recommendations for volunteer management.

---

## Objectives

- Analyze volunteer participation data.
- Identify trends and patterns in volunteer engagement.
- Visualize volunteer activity using charts and graphs.
- Predict future volunteer participation using Machine Learning.
- Provide AI-based recommendations for volunteer retention and engagement.

---

## Dataset Features

The dataset contains the following attributes:

| Feature | Description |
|----------|------------|
| Volunteer_ID | Unique volunteer identifier |
| Age | Age of volunteer |
| City | Volunteer's city |
| Event | Event participated in |
| Hours_Contributed | Total hours contributed |
| Previous_Participation | Number of previous participations |
| Training_Attended | Whether training was attended |
| Participated_Next_Event | Target variable for prediction |

## Machine Learning Model

### Objective

Predict whether a volunteer is likely to participate in future events.

### Algorithm Used

- Logistic Regression

### Features Used

- Age
- Hours Contributed
- Previous Participation
- Training Attended

### Evaluation Metrics

- Accuracy Score
- Confusion Matrix

## Key Insights

### Volunteer Engagement

- Volunteer participation varies across cities.

### Machine Learning Findings

### AI Findings
